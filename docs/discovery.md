# NINJA Discovery

Gate: G0 Idea & Value
Status: PASS
Date: 2026-05-13

## User / Persona
Vikas uses Windows 11 and wants a personal desktop AI assistant that stays available in the background. He is technical enough to run npm commands and configure API keys, but the app must behave like a polished desktop product rather than a developer demo.

## Pain Point
Vikas needs a single always-available assistant that can listen, respond by voice, remember context, execute safe local tools, and surface important briefings without requiring him to keep a browser tab open.

## Success Metric
Phase 1 succeeds when NINJA can run as an Electron desktop app, show an ambient widget, open with Alt+N, accept typed and voice input, answer through OpenCode Go/Kimi when keys are configured, speak through ElevenLabs when keys are configured, read/write local memory, and enforce GREEN/YELLOW/RED tool permissions.

## MVP vs Future
MVP Phase 1 includes Electron shell, tray, widget, hotkey, active panel, text chat, voice input/output plumbing, OpenCode-backed brain, memory files, core tools, conscience layer, morning notification, deep-work UI, and optional Porcupine wake-word startup.

Future Phase 2 includes HeyGen avatar, WhatsApp, email/calendar, full Playwright browser control, gbrain memory upgrade, and self-improvement log processing.

## Out Of Scope
Phase 1 does not include HeyGen, WhatsApp, email/calendar, trading execution, financial actions, or hidden destructive automation.

## Value Proposition
NINJA turns the desktop into a persistent personal AI control room that can talk, remember, and act with explicit safety boundaries.
